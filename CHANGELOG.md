# Changelog

## 0.2.0 — 2026-05-29

### Added
- **Slides** — create slide decks, multi-page editor, text boxes, shapes, and notebook support
- **Whiteboards** — drawing tools, pen colors/sizes, shapes, text boxes, and notebook support
- **Home workspace** — Notes, Slides, and Whiteboards tabs with search
- **Notebooks** — popup view, mixed notes/slides/whiteboards, remove items from notebooks
- **Trial mode** — try the app without an account (local-only, no save)
- **Account deletion** — Settings → Delete account (Supabase setup required)
- **Search** — filter notes, slides, whiteboards, and notebooks from the home page

### Improved
- Note and slide editors — click-to-open dropdowns, heading/text formatting fixes
- Slide editor — scrollable thumbnails, next slide, right-click delete slide page
- Home and notebook card layouts

## 0.1.0

- Initial release — notes, notebooks, themes, Supabase sign-in
